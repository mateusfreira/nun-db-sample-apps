NunDB Sample Applications

Real-world examples demonstrating NunDB capabilities

Explore interactive examples that showcase NunDB's real-time synchronization, collaborative features, and persistent storage capabilities. Each application is designed to run directly in your browser with zero setup required.

PS: All this applications are example of AI assisted development using NunDB as a database. It is not mine and I have not curated it as in any other project of mine I would. You can use them as examples or as a starting point for your own projects. Most of them are projects where I wanted to see how the AI tools would behave when reasoning about NunDb APIs documentation. If you want to see better code examples to follow please go check https://github.com/mateusfreira/nun-db-js/tree/master/examples

A collaborative time tracking application that demonstrates real-time synchronization across multiple users. Perfect for teams who need to track time spent on tasks with live updates and workspace isolation.

🦷 Dental Odontogram

A canvas-based dental odontogram with realistic mouth-shaped arch layout. Click teeth to mark statuses like cavity, crown, implant, or missing, add clinical notes, and sync everything in real-time via NunDB.

  • Realistic arch-shaped tooth layout (32 teeth)
  • 7 dental statuses with color-coded visualization
  • Per-tooth clinical notes
  • Real-time sync across tabs via NunDB watch
